|---|---|

| **Distance** | ~200 km across Strait of Hormuz |
| **Duration** | 7-12 hours (overnight) |
| **Operator** | Al Hili Marine Services |
| **Schedule** | Sun, Tue, Thu at 9:00 PM from Sharjah |

### 2. Chittagong → Laem Chabang
**Trip 2→3 | December 2027 | Bay of Bengal → Gulf of Thailand**

| | |
|---|---|
| **Why** | Myanmar borders closed to overlanders |
| **Distance** | ~3,500 km via Strait of Malacca |
| **Duration** | 10-14 days |
| **Method** | RoRo or Container shipping |
| **Operators** | K-Line, freight forwarders |
| **Est. Cost** | **$2,500** |

**Route:**
Chittagong → Bay of Bengal → Andaman Sea → Strait of Malacca → Singapore Strait → South China Sea → Gulf of Thailand → Laem Chabang

**Key Notes:**
- Major shipping route with regular service
- K-Line Thailand acts as Asian Regional HQ for RoRo
- Laem Chabang is Thailand's main vehicle import port (near Bangkok)
- Allow 2-3 weeks total including paperwork

**Process:**
1. Contact freight forwarder in Chittagong
2. Arrange Thai temporary import permit
3. Ship vehicle, fly to Bangkok
4. Clear customs at Laem Chabang

**Sources:** [K-Line Thailand](https://www.kline.co.th/services/roro/), [All Transport Depot](https://alltransportdepot.com/ship-car-to-laem-chabang-thailand/)

### 4. Merak → Bakauheni (Sunda Strait Ferry)
**Trip 4 | November 2028 | Java to Sumatra**

| | |
|---|---|
| **Why** | Crossing between Java and Sumatra |
| **Distance** | ~30 km across Sunda Strait |
| **Duration** | 1.5-2.5 hours |
| **Operator** | ASDP Indonesia Ferry |
| **Schedule** | Every 12 minutes, 24/7 operation |
| **Vehicle Cost** | Rp 343,000 (~$22) |
| **Est. Cost** | **$50** (round trip with passengers) |

**Key Notes:**
- The **only way to get a car to Sumatra** from Java
- Very frequent departures — just show up and queue
- Efficient, well-run ferry service
- Book Merak (west Java) ↔ Bakauheni (south Sumatra)

**Route Plan:**
1. From Jakarta, drive west to Merak (~100 km, 3 hrs)
2. Ferry to Bakauheni
3. Drive north through Sumatra (Lampung → Palembang → Jambi → Padang → Bukittinggi → Lake Toba → Medan)
4. Return south on same route
5. Ferry back to Merak
6. Continue east through Java

**Tips:**
- No advance booking needed
- Pay at terminal
- During holidays (Lebaran) expect queues of 6-12 hours

**Sources:** ASDP Indonesia Ferry (indonesiaferry.co.id)

### 5. Dili → Darwin
**Trip 4 | January 2029 | Timor Sea Crossing**

| | |
|---|---|
| **Why** | Continuing from Timor-Leste to Australia |
| **Distance** | ~700 km across Timor Sea |
| **Duration** | 2 days voyage (10 days cycle) |
| **Operators** | Swire Shipping, Toll Group (Perkins) |
| **Est. Cost** | **$1,500** |

**Key Notes:**
- **No passenger ferry exists** — freight shipping only
- Swire Shipping East Timor Service runs every ~10 days
- Route: Singapore → Dili → Darwin → Surabaya → Singapore
- You fly separately and meet vehicle in Darwin
- Australian biosecurity is strict — vehicle must be spotlessly clean

**Process:**
1. Contact Swire Shipping or Toll Group
2. Arrange Australian temporary import (Carnet de Passage accepted)
3. Ship vehicle from Dili
4. Fly to Darwin
5. Clear quarantine/customs (allow extra days)

**Sources:** [The Way Overland](https://www.thewayoverland.com/post/shipping-ferries), [Swire Shipping](https://www.swireshipping.com/information/latest-news/2023-03-new-east-timor-services/)

### 9. Cartagena → Panama City (Darién Gap)
**Trip 8 | May 2031 | RORO + San Blas Sail**

| | |
|---|---|
| **Why** | No road through Darién Gap |
| **Distance** | ~500 km by sea (avoiding 100 km roadless jungle) |
| **Duration** | Vehicle: 1-2 days | You: 5 days (San Blas sail) |
| **Method** | Vehicle via RORO, passengers via sailboat |
| **Est. Cost** | **$2,150-2,650** (vehicle) + **$650 pp** (sailing) |

**Recommended Approach: RORO + San Blas Sail**

Ship the car separately while you sail through the San Blas Islands — an incredible experience rather than just waiting for your vehicle!

---

**STEP 1: Book RORO Shipping (3-4 weeks before)**

**Agent:** IVSS UK (ivssuk.com)
- UK: +44 203 787 4201 | US: +1 917-781-4536
- Email: info@ivssuk.com

**Costs:**
- ~$500 minimum freight
- Port fees + agent fees + customs
- **Total: ~$1,500-2,500**

**Requirements:**
- Vehicle title/registration (originals + copies)
- Passport + TIP (Temporary Import Permit)
- Police inspection in Panama City (5+ days before, arrive 6am)
- Max ¼ tank fuel
- Remove/disconnect lithium batteries (ship separately)
- Leave keys with port agent
- Closed-toe shoes, long pants, long sleeves at port

---

**STEP 2: Book San Blas Sailing**

**Operator:** Wild Card Sailing (wildcardsailing.com)
- Cost: ~$650 USD pp (includes San Blas tax)
- Duration: 5 days total
  - 3 days island-hopping in San Blas
  - 2 days open sea to Cartagena
- Route: Puerto Lindo (Panama) → Cartagena
- Book to arrive Cartagena ~same time as vehicle

**What to Bring:**
- Dramamine (it can get rough!)
- Cash for balance payment
- Light backpack only (no suitcases)

---

**STEP 3: Collect Vehicle in Cartagena**
- Get notarized power of attorney for agent (they email form)
- Wear closed-toe shoes, long pants, long sleeves to port
- Pay port fees
- Get Colombian TIP + mandatory insurance
- Bring tire compressor + fuel money
- Allow 3-5 working days after ship arrival

---

**Key Contacts:**
- **IVSS UK:** info@ivssuk.com | +1 917-781-4536
- **Wild Card Sailing:** wildcardsailing.com
- **Backup agent:** Overland Embassy +507 62993540

**Buffer:** Plan 3 weeks total for the entire process

**Sources:** [IVSSUK](https://ivssuk.com/shipping-across-the-darien-gap-colombia-to-panama/), [Wild Card Sailing](https://wildcardsailing.com)

### 15. Brazzaville → Kinshasa
**Trip 19 | July 2040 | Congo River Crossing**

| | |
|---|---|
| **Why** | Crossing from Congo-Brazzaville to DRC |
| **Distance** | ~4 km across the Congo River |
| **Duration** | 20 minutes |
| **Operators** | Various local ferry operators |
| **Schedule** | Multiple daily crossings |
| **Est. Cost** | **$50** (vehicle + passengers) |

**Key Notes:**
- Shortest capital-to-capital distance in the world
- Brazzaville (Republic of Congo) and Kinshasa (DRC) face each other across the river
- Ferry departs from Beach Ngobila port in Kinshasa / similar in Brazzaville
- Essential crossing for the Africa route avoiding conflict zones

**Process:**
1. Clear exit immigration in Brazzaville
2. Board ferry at port
3. 20-minute crossing
4. Clear DRC immigration and customs in Kinshasa
5. Drive west through DRC's Kongo Central province to Matadi, then to Angola

**Requirements:**
- DRC visa (arrange in advance)
- Vehicle Carnet de Passage
- Expect delays and informal "fees" — patience required
- Allow half day for the crossing including paperwork

**Tips:**
- Use a fixer/guide familiar with the crossing
- Have small bills ready
- Early morning crossings tend to be less chaotic
