# A Grand Tour

**One Family. One Land Cruiser. 100 Countries.**

A 15-year around-the-world overland expedition (2027-2042) circumnavigating the globe by road — through the mountains of Pakistan, the temples of Southeast Asia, the outback of Australia, the length of the Americas from Patagonia to Alaska, across Europe, and down through Africa.

| | |
|---|---|
| **Duration** | Feb 2027 – Aug 2042 |
| **Distance** | ~200,000 km |
| **Countries** | 100+ |
| **Vehicle** | Toyota Land Cruiser 300 |

## Route

```
LEG 1: EAST (2027-2029)
Dubai → Iran → Pakistan → India → SE Asia → Australia → New Zealand

LEG 2: AMERICAS (2030-2033)
Chile → Argentina → Brazil → Peru → Central America → USA → Canada → Alaska

LEG 3: WEST (2034-2042)
UK → Europe → Nordics → Balkans → Caucasus → Morocco → West Africa →
Central Africa → Southern Africa → East Africa → Egypt → Middle East → Dubai
```
